# v2ray-SiteDAT

## 自定义site.dat 文件

### 下载

  - https://github.com/onplus/v2ray-SiteDAT/releases

### 使用：

  [1]  - 修改/添加sites文件夹下的域名文件，运行 ./v2sitedat 生成geosite.dat
    
       - 更多参考 ./v2sitedat --help
    
  [2]  - 可Fork后自行在sites文件夹下增减,后每天自动Build & Release
  

**改自 https://github.com/v2ray/ext/tree/master/tools/geosites**
